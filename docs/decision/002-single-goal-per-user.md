# 002 — Single Goal Per User

**Date:** 28 Feb 2026
**Status:** Decided

## Decision
User can only have one active goal at a time. They can change it anytime in settings.

## Why
Optimizing for everything simultaneously means improving nothing. Forcing one goal creates intentionality. Seasonal goal switching — bulking in winter, cutting in summer — is a feature, not a limitation.

## Alternatives Considered
Multiple simultaneous goals — rejected. Nutrient priority conflicts between goals make the dashboard logic ambiguous and the user experience confusing.

## Consequences
Need "change goal" option in user settings. When goal changes, dashboard reorients immediately.
